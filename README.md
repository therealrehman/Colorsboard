# ColorKey Keyboard

A vibrant, customizable Android keyboard inspired by Samsung KeysCafe.

## Features

- 8 Themes (Classic, Neon, Pastel, Ocean, Sunset, Midnight, Rainbow, Cyber)
- Master Theme with radial color animation on every key press
- 5 Key Effects (Ripple, Particle, Radar, Bloom, Shockwave)
- 6 Sound Types (Mechanical, Typewriter, Bubble, Wood, Digital, Soft)
- Volume control + Vibration intensity
- Key Blending effect
- Board Blending effect
- Key Motion animation
- Emoji Picker with 96 emojis
- Glassmorphic UI design

## Tech Stack

- Kotlin
- Material3 Components
- Android SoundPool
- SharedPreferences + Gson

## Inspired By

Samsung KeysCafe (samboard) - All code written from scratch.
